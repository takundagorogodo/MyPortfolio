# 🌐 Personal Portfolio Website

A modern and fully responsive **Portfolio Website** built to showcase my skills, projects, and experience as a **Full Stack Developer**.

This website serves as my digital presence where recruiters and developers can explore my work, technical skills, and contact information.

---

## 📌 Project Overview

The portfolio is designed with a focus on **clean UI, smooth animations, and responsiveness** across all devices.

It highlights:
- Technical skills
- Featured projects
- Education background
- Contact details

---

## 🚀 Features

- 📱 Fully Responsive Design  
- 🌙 Dark / Light Mode Toggle  
- ⚡ Smooth Scroll Animations (AOS)  
- 🧭 Sticky Navigation Bar  
- 📂 Projects Showcase  
- 🧠 Skills Section with Icons  
- 🎓 Education Timeline  
- 📬 Contact Section UI  
- 🔗 Social Media Integration  

---

## 🛠️ Technologies Used

- HTML5  
- Tailwind CSS  
- JavaScript (Vanilla JS)  
- AOS (Animate On Scroll)  
- Font Awesome  
- Devicon  

---
## 🎯 Purpose of the Project

This portfolio was built to:

Showcase my development skills
Present my projects professionally
Create a strong online presence
Make it easy for recruiters to connect with me
